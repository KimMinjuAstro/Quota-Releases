# Quota

Gemini · Claude · ChatGPT 사용량을 한 화면에서 실시간으로 보여주는 Always On Top 데스크톱 위젯입니다.

이 문서는 QA로 테스트해주실 분들을 위한 **설치·사용 안내**입니다.

## 1. 다운로드

1. [Releases 페이지](https://github.com/KimMinjuAstro/Quota-Releases/releases) 로 이동
2. 가장 위에 있는 최신(Latest) 버전 클릭
3. `Assets` 목록에서 `Quota_x.x.x_x64-setup.exe` (또는 `.msi`) 파일 다운로드
4. 브라우저(Edge 등)가 다운로드 시점에 **"일반적으로 다운로드되지 않는 파일"** 경고를 띄울 수 있습니다.
   많이 안 퍼진 개인 배포 앱이라 뜨는 정상적인 경고예요. → **"유지"** 클릭해서 다운로드 완료

## 2. 설치

1. 다운로드한 설치 파일을 더블클릭 실행
2. Windows가 **"Windows에서 PC를 보호했습니다" (알 수 없는 게시자)** 경고를 띄울 수 있습니다.
   개인 배포 앱이라 서명 인증서가 없어서 뜨는 정상적인 경고예요.
   → **"추가 정보"** 클릭 → **"실행"** 버튼 클릭
3. 설치 마법사 따라 진행하면 완료. 설치가 끝나면 Quota가 자동으로 실행됩니다.
   - 컴퓨터에 WebView2가 없으면 설치 중 자동으로 내려받는데, 이 구간에서 진행 바가 잠깐 멈춘 것처럼 보일 수 있어요 — 얼어붙은 게 아니라 정상 동작이니 잠시 기다려주세요.

## 3. 처음 사용하기

- 실행하면 화면 한쪽에 작은 카드 모양 위젯이 뜹니다. **항상 다른 창 위에 떠 있어요.**
- 위젯 어디든 눌러서 드래그하면 원하는 위치로 옮길 수 있습니다.
- 헤더의 버튼들:
  | 아이콘 | 기능 |
  |---|---|
  | ⚙ | 설정 메뉴 (다크/라이트 테마, 위젯 확대·축소) |
  | ＋ | Gemini/Claude/ChatGPT 계정 추가 |
  | ⟳ | 새로고침 |
  | ✕ | 종료 |
- **계정 추가**: ＋ 버튼 → Gemini/Claude/ChatGPT 선택 → 로그인 창이 뜨면 평소 쓰는 계정으로 로그인
  - 로그인 처리 중엔 카드에 "로그인 중…" 표시가 뜨고 아이콘이 돌아갑니다. 로그인 완료까지 몇 초 걸리는 건 정상입니다.
- 로그인이 끝나면 카드에 **현재 세션 / 주간 한도** 사용률(%)과 다음 리셋까지 남은 시간이 뜹니다.
- 카드의 ✎(별명 수정) / ↪(로그인/로그아웃) / 🗑(계정 제거) 버튼도 한 번씩 눌러봐 주세요.
- 카드를 드래그하면 순서를 바꿀 수 있습니다.
- 위젯 오른쪽/아래 가장자리에 열/행 개수를 조절하는 화살표 버튼이 있습니다 — 카드가 몇 개든 원하는 격자 모양으로 배치할 수 있어요.
- ⚙ 설정 메뉴의 −/＋ 버튼으로 위젯 전체 크기를 확대·축소할 수 있습니다.

## 4. 이런 부분 위주로 테스트해주세요

- 계정 추가 → 로그인 → 사용량이 뜨기까지 전체 흐름이 매끄러운지 (막히거나 멈추는 곳 있는지)
- 여러 계정(Gemini 여러 개, Claude 2개까지, ChatGPT 1개)을 동시에 등록했을 때 데이터가 서로 안 섞이는지
- 다크/라이트 테마 전환, 창을 옮긴 뒤 앱을 재시작해도 위치가 기억되는지
- 카드 드래그 순서 변경, 열/행 조절, 확대/축소가 재시작 후에도 그대로 유지되는지
- 컴퓨터를 오래 켜둔 상태에서도 사용량이 계속 잘 갱신되는지 (30초 주기)
- 이스터에그 — 재미로 한번 찾아보셔도 좋아요

## 5. 이미 알고 있는 이슈

- Claude 로그인 팝업이 완료 후 자동으로 안 닫히고 남아있는 경우가 있음
- Claude 두 번째 계정 카드의 ↪(로그인 창 열기) 버튼은 동작하지 않음
- ⟳ 새로고침 버튼은 Claude 두 번째 계정에는 적용되지 않음

## 6. 버그 발견 시 / 피드백

발견한 버그, 이상한 화면, 하고 싶은 말은 **디스코드 또는 카카오톡**으로 편하게 알려주세요.

캡처와 함께 "무엇을 눌렀는지 → 무슨 일이 일어났는지"만 같이 적어주시면 큰 도움이 됩니다.

## 7. 개인정보 관련 안내

- 구글/Claude 로그인은 각자의 계정으로 **직접** 로그인하는 것이며, 아이디·비밀번호는 앱이나 개발자에게 전달되지 않습니다.
- 계정 별명·순서 등 설정값은 내 컴퓨터의 `%APPDATA%\com.astrominju.quota\config.json`에만 저장되고 외부로 전송되지 않습니다.

## 8. 제거(삭제)하는 법

- Windows 설정 → 앱 → **Quota** 찾아서 제거
- 계정/설정 데이터까지 완전히 지우려면 제거 후 `%APPDATA%\com.astrominju.quota` 폴더도 삭제
